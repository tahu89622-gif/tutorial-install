Tutorial Lengkap: Cara Install Kali Linux di VirtualBox

Penulis: Awal Marudut Gultom | Junior IT Support & Flutter Developer

Kenapa Pakai VirtualBox?

Untuk belajar Cybersecurity dan menggunakan tools profesional seperti Nmap atau Ghidra, lingkungan VirtualBox adalah media paling aman untuk eksperimen.

Langkah Instalasi

  1. Buka software VirtualBox dan klik tombol "New".
  2. Pilih tipe Linux dan versi Debian (64-bit) agar kompatibel dengan Kali Linux.
  3. Pilih file ISO Kali Linux yang sudah Anda download.
  4. Klik "Start" dan ikuti petunjuk Graphical Install hingga selesai.

Optimasi Pasca Instalasi

Agar sistem selalu aman dan tools diperbarui, jalankan perintah berikut di terminal:

sudo apt update && sudo apt upgrade -y

Pastikan koneksi internet stabil saat melakukan pembaruan sistem.
